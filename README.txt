Project Name: Library Management System
Author: Shaaz Saleem
Date: April 7, 2024

Description:
The Library Management System is a console-based application designed to manage a library's book 
catalog and user registry. 
It allows librarians to add, remove, and search for books, 
as well as register new users and manage book borrowing and returning.

Installation:
1. Download all files containing the whole project to computer
2. Open the files using your preferred environment
3. Run the code in the LibraryManagementSystem.java file
4. Follow the instructions in the console

Usage:
1. Follow the on-screen prompts to navigate the menu and interact with the Library Management System.

Features:
- Book Management: Add and remove books from the library's catalog.
- User Registration: Register new users with unique library IDs.
- Book Borrowing: Allow users to borrow books from the library.
- Book Returning: Enable users to return borrowed books.
- Display Borrowed Books: View the list of books borrowed by a specific user.
- Search for Books: Search for books in the library's catalog.
